#### Resumo condicionais no Shell

* Comparação de strings:

= (Verificar igualdade)

!= (Verificar se é diferente)

-n (Verifificar se é não nula)

-z (Verificar se é nula)

* Comparação numérica:

-lt ( < Menor) 

-gt ( > Maior)

-le ( <= Menor ou igual)

-ge ( >= Maior ou igual)

-eq ( = Igual)

-ne (!= Diferente)
